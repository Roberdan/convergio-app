# MnAdminShell

Admin layout shell with sidebar navigation, header, and content area.

## When to use

Use as the top-level layout for admin or back-office applications.

## Example

```tsx
import { MnAdminShell } from "@/components/maranello"

<MnAdminShell />
```

## Accessibility

- Uses semantic HTML elements for proper document structure
- Landmarks (`<main>`, `<nav>`, `<aside>`) structure the layout
- Skip-to-content link is supported
