# MnGridLayout

Responsive grid layout manager with drag-and-drop widget repositioning.

## When to use

Use when users need to rearrange widgets within a grid.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `columns` | `GridColumns` | — | — |
| `breakpoints` | `Breakpoints` | — | — |

## Example

```tsx
import { MnGridLayout } from "@/components/maranello"

<MnGridLayout />
```

## Accessibility

- Uses semantic HTML elements for proper document structure
- Landmarks (`<main>`, `<nav>`, `<aside>`) structure the layout
- Skip-to-content link is supported
