# MnA11yFab

Floating action button toggling accessibility overlays and settings.

## When to use

Use to give users quick access to accessibility tools from any page.

## Props

_This component extends native HTML attributes. See source for full type details._


## Example

```tsx
import { MnA11yFab } from "@/components/maranello"

<MnA11yFab />
```

## Accessibility

- Uses semantic HTML elements for proper document structure
- Theme changes are announced via `aria-live`
- All controls meet WCAG 2.1 contrast requirements
