# MnA11yFab

Floating action button toggling accessibility overlays and settings.

## When to use

Use to give users quick access to accessibility tools from any page.

## Props

_This component extends native HTML attributes. See source for full type details._


## Example

```tsx
import { MnA11yFab } from "@/components/maranello"

{/* Add once in your root layout */}
<MnA11yFab />
```

## i18n

All labels are translatable via `MnLocaleProvider` or the YAML `locale:` section.

```tsx
import { MnLocaleProvider } from "@/lib/i18n"

const esLabels = {
  a11yFab: {
    accessibilitySettings: "Configuración de accesibilidad",
    textSize: "Tamaño de texto",
    reducedMotion: "Movimiento reducido",
    highContrast: "Alto contraste",
    focusIndicators: "Indicadores de enfoque",
    resetToDefaults: "Restablecer valores predeterminados",
  },
};

<MnLocaleProvider messages={esLabels}>
  <MnA11yFab />
</MnLocaleProvider>
```

Or via YAML config:

```yaml
locale:
  a11yFab:
    accessibilitySettings: "Configuración de accesibilidad"
    textSize: "Tamaño de texto"
```

## Accessibility

- Theme changes are announced via `aria-live`
- All controls meet WCAG 2.2 AA contrast requirements
